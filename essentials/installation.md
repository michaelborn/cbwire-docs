# Installation

Install [CommandBox](https://www.ortussolutions.com/products/commandbox).

Within the root of your project, run:

```bash
box install cbwire
```

If you want the latest bleeding-edge, run:

```bash
box install cbwire@be
```

## Requirements

* Adobe ColdFusion 2018+ or Lucee 5+
* ColdBox 6+
